# Game Setup

## Board Setup

- Place market deck in center
- Create space for 6 face-up market cards
- Each player creates their play area with:
  - Defense rows (0-3)
  - Draw/discard piles
  - General zone
  - Hand area

## Starting State

- Each player starts with 40 HP
- Shuffle 8-card starting deck
- Place Kingsguard in Row 0
- First player draws 3 cards
- Other players draw 4 cards
